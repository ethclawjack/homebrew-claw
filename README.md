# Homebrew Tap for claw-relay

Homebrew tap for installing [claw-claude-code-relay](https://github.com/ethclawjack/claw-claude-code-relay).

## Installation

```bash
brew tap ethclawjack/claw
brew install claw-relay
```

## Usage

```bash
claw-relay --help
claw-relay "Build an API"
```

For full documentation, visit: https://ethclawjack.github.io/claw-claude-code-relay/
